# MediaNotes

Подключаемый модуль для создания заметок (скетч/текст/фото/аудио).

`audio-recorder` — модуль для записи и воспроизведения аудио.
`media-notes-compose` — основной модуль библиотеки.
`sampleApp` — сэмпл Android-приложения.
